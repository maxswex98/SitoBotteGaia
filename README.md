# Botte Gaia

Sito statico prototipale per il ristorante Botte Gaia a Reggio Emilia.

## Contenuto

- `index.html`: pagina pubblica del sito
- `admin.html`: area admin locale per aggiornare testi e menu nel browser
- immagini locali del ristorante e dei piatti

## Pubblicazione

Il progetto e statico, quindi puo essere pubblicato facilmente su:

- GitHub Pages
- Netlify
- Vercel

Per la pubblicazione della sola vetrina pubblica, il file di ingresso e `index.html`.
